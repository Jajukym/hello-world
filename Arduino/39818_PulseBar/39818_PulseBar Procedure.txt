ETFM39818 Pulse Bar Test

Tools:

ESD Strap

Nitrile Gloves

Arduino Uno R3

Arduino IDE

USB Cable

Pulse Bar Harness



One time Arduino Setup:

Download latest Arduino IDE (1.8.5) and install from https://www.arduino.cc/en/Main/Software

Open Arduino IDE

Paste provided folder into: C:...\Documents\Arduino

Open provided folder and click on 39818_PulseBar.ino file and Arduino IDE will open

Connect the Arduino to the PC with USB cable

In the upper left of the IDE: click the arrow icon. The code will compile and upload to Arduino

Connect the harness to the Arduino

Disconnect the USB from Arduino


Setup:
Open: C:...\Documents\Arduino\39818_PulseBar and click file in folder. The Arduino IDE will open


Test:

Ensure the unit is packaged in an antistatic bag

Verify that the ESD bag is dissipative and it is printed on as per the drawing

Visually inspect the pulse bar for cosmetic defects, physical damage, and illegible printing

Connect harness to pulse bar

Plug the USB into the Arduino

In the upper right of the IDE: click the magnifying glass and a terminal will appear

Insert and remove DMK and verify screen responds accordingly

Press each button on the pulse bar and verify screen responds accordingly

Hold hand pulse plates and a reading will appear

Turn on Polar pulse simulator or gently rub the Polar chest pulse sensors and a reading will appear

Disconnect the USB from Arduino

Disconnect harness from sensor

Repeat
